Overview
--------

Pyway is a database versioning and migration tool inspired on Flyway

Installation / Usage
--------------------

To install use pip:

    $ pip install pyway


Or clone the repo:

    $ git clone https://github.com/sergiosbx/pyway.git
    $ python setup.py install
